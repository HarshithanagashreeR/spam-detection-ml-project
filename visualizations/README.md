# Visualizations

This folder contains all visualizations generated by the spam detection system.

## Files:
- `class_distribution.png` - Pie and bar charts showing ham vs spam distribution
- `message_statistics.png` - Box plots of message length and word count
- `word_clouds.png` - Word clouds comparing spam and ham vocabulary
- `model_comparison.png` - Bar chart comparing model accuracies
- `confusion_matrix.png` - Heatmap showing prediction accuracy
- `roc_curves.png` - ROC curves for all models
